# Roadmap: 자연어 → Flow 자동 생성

## 개요

사용자가 자연어 명령을 입력하면 자동으로 Workflow(Flow)를 생성하는 기능이다.
비개발자의 자동화 접근성을 한 단계 더 높이는 것을 목표로 한다.

---

## 배경

FlowKit은 Low-code 플랫폼으로서 비개발자를 위한 자동화를 지향한다.
현재는 Step, Condition, Approval 중심의 UI 기반 흐름 정의를 제공하지만,
자연어 기반 인터페이스를 통해 더 직관적인 자동화 정의가 가능하다.

---

## 목표

- 자연어 명령 → 자동화 Flow 생성
- 실행 데이터 기반 자동화 추천
- 조직 단위 템플릿 및 표준화

---

## 예상 기능

### 1. 자연어 → Flow 변환

- 사용자 입력: "매주 월요일에 팀 회의록을 Notion에 생성하고 Slack에 알림 보내줘"
- 시스템 출력: 스케줄 트리거 → Notion 페이지 생성 Step → Slack 알림 Step으로 구성된 Workflow

### 2. 실행 데이터 기반 추천

- 기존 실행 데이터를 분석하여 자주 사용되는 패턴 추천
- 유사 Workflow 제안 및 템플릿 추천

### 3. 조직 단위 템플릿

- 팀/조직 단위로 표준화된 Workflow 템플릿 제공
- 템플릿 기반 빠른 Workflow 생성

---

## 기술 검토 사항

- LLM(GPT 등) 기반 자연어 해석
- Workflow DSL → DAG 변환 파이프라인
- 생성된 Flow의 검증 및 사용자 확인 단계
- 오류/모호성 처리 전략

---

## 범위

- 초기 버전에서는 Non-Goal로 분류됨
- 핵심 플랫폼 안정화 이후 확장 단계에서 진행 예정
